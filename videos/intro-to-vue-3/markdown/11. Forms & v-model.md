## Introducing v-model

At the beginning of this course, we learned about `v-bind`, which creates a one-way binding, from the data to the template. When working with forms, however, this one way binding isnâ€™t enough. We also need to be binding from the template to the data.

For example, when a user inputs their name into an input field, we want to record and store that value in our data. The `v-model` directive helps us achieve this, creating two-way data binding.
